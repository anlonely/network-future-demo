# Network Future Demo

Public-safe React/Vite portfolio demo extracted from a deployed personal site and rewritten with sanitized content.

## What changed

- Replaced personal domains, email, and branding with demo-safe placeholders
- Swapped outbound links to public GitHub destinations
- Rewrote portfolio card copy to example narratives
- Removed unrelated runtime dependencies so the repo is easier to publish and run

## Stack

- React 19
- Vite 6
- Tailwind CSS v4
- TypeScript
- Lucide React

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This repository is intentionally sanitized for public storage.
- Any references to infrastructure, metrics, or links are demo placeholders rather than production data.
